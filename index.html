Privacy Policy

This application uses the WhatsApp Cloud API for testing and demonstration purposes. It enables the sending and receiving of messages through a connected WhatsApp Business Account.

No personal data is stored, shared, or sold. Messages are processed solely for testing and functional demonstration. Once testing is complete, all temporary data is discarded.
